The data about BBCNews dataset is from link url = {http://mlg.ucd.ie/datasets/bbc.html} by clicking "Download raw text files" in Dataset: BBC.
Decompressed "bbc-fulltext.zip" includes five folders for 5 categories and the "README.TXT". The five folders for 5 categories are in folder "data\bbcnews" and the "README.TXT" is stored in folder "data".

## code

In "code" folder, there are five folders for the BBCNews dataset about document categorization.

## figure

"figure" folder are Figures created by the above codes and used in the article.

## created data

"created data" folder is to store data created by above codes.
"created data\create matrix" is used in "code\BBCNews create data" folder. 
"created data\centroid linkage", "created data\average linkage", "created data\single linkage", and "created data\complete linkage" are used in codes with "code\BBCNews analyze data" and "code\BBCNews plot data" folders.
