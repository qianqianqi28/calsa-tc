The data about 20 Newsgroups dataset is from link url = {http://www.qwone.com/~jason/20Newsgroups/} by clicking "20news-bydate.tar.gz" in Data. Decompressed "20news-bydate.tar.gz" includes "20news-bydate-train" folder and "20news-bydate-test" folder.
In this research, we use documents from "comp.graphics", "rec.sport.hockey", and "sci.crypt". These documents are stored in "data\20news-bydate-train\used for experiments" folder and "data\20news-bydate-test\used for experiments" folder.
The "data\20news-bydate-train\used for experiments" folder is the documents from "comp.graphics" (584 documents), "rec.sport.hockey" (600 documents), and "sci.crypt" (595 documents) of "20news-bydate-train" folder. 
The "data\20news-bydate-test\used for experiments" folder of "20news-bydate-test" folder is the documents from "comp.graphics" (389 documents), "rec.sport.hockey" (399 documents), and "sci.crypt" (396 documents) of "20news-bydate-test" folder.
